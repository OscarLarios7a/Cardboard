Prueba para la empresa Cardboard México

El siguiente Codigo ha sido desarrollado a manera de prueba, según lo acordado via E-mail, 
está desarrollado en laravel V 5.4, la maquetación esta realizada con bootstrap, utiliza la libreria
formValidation para validar los inputs de los formularios.

## Instalacion y Uso

<p>Clona el proyecto</p>
<p>Dirigete a la raiz del proyecto mediante la consola</p>
<p>Ejecuta: composer install</p>
<p>Ejecuta: php artisan migrate</p>
<p>Ejecuta: php artisan db:seed</p>
<p>Ejecuta: php artisan serve (dirigirse a http://localhost:8000/)</p>
<p>NOTA: Por favor agregar las credenciales de conexion y de envio de email en el archivo .env</p>
</br>
